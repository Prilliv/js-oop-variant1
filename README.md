# js-oop-variant1

## Тема
Система управління транспортом.

## Опис
Проєкт реалізує систему управління автопарком за допомогою JavaScript ES6 класів.

У програмі використано:
- ES6 класи;
- конструктори;
- наслідування `extends`;
- виклик батьківського конструктора через `super`;
- приватні поля `#`;
- статичні методи;
- інкапсуляцію;
- наслідування;
- поліморфізм.

## Класи

### Vehicle
Базовий клас для всіх транспортних засобів.

Поля:
- brand;
- model;
- year;
- mileage;
- fuelLevel.

Методи:
- `drive(distance)` — збільшує пробіг і зменшує паливо;
- `refuel(amount)` — заправляє транспорт;
- `getMaintenance()` — розраховує вартість обслуговування;
- `getInfo()` — повертає інформацію про транспорт;
- `getType()` — повертає тип транспорту.

### Car
Клас легкового автомобіля. Наслідується від `Vehicle`.

Додаткові поля:
- doors;
- transmission.

### Truck
Клас вантажівки. Наслідується від `Vehicle`.

Додаткові поля:
- loadCapacity;
- isLoaded.

### Motorcycle
Клас мотоцикла. Наслідується від `Vehicle`.

Додаткові поля:
- type.

### Fleet
Клас для управління автопарком.

Методи:
- `addVehicle(vehicle)` — додає транспорт;
- `removeVehicle(index)` — видаляє транспорт;
- `getAverageMileage()` — рахує середній пробіг;
- `getTotalMaintenanceCost()` — рахує загальну вартість обслуговування.

Статичні методи:
- `filterByType()`;
- `filterByYear()`;
- `filterByMileage()`.

## Функціонал
- додавання автомобіля, вантажівки або мотоцикла;
- відображення списку транспорту;
- видалення транспорту;
- поїздка на 50 км;
- заправка;
- фільтрація за типом;
- фільтрація за роком;
- фільтрація за пробігом;
- статистика автопарку.
